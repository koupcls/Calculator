> Интерактивный интсрумент для изучения алгоритмов кодирования, шифрования и сжатия данных

🔗 **[Открыть приложение](https://koupcls.github.io/Calculator)**

---

## ✨ Возможности

### Кодирование (Huffman / Shannon-Fano)
- Расчёт энтропии, избыточности и эффективности источника
- Построение деревьев кодирования с визуализацией
- Таблицы кодов и математическое ожидание длины кода
- Пошаговые формулы расчёта метрик

### Шифрование (дорабатывается)
- **Шифр Виженера**: многоалфавитная замена с ключом
- **Перестановка с ключом**: колоночная транспозиция с визуализацией таблицы
- Цепочки операций: последовательное применение шифров
- Интерактивный редактор алфавита с плитками

### Сжатие (LZ-семейство) (вообще плохо работает)
| Алгоритм | Формат кода | Описание |
|----------|-------------|----------|
| **LZ77** | `<сдвиг, длина, символ>` | Скользящее окно, триплеты |
| **LZSS** | `0, 'символ'` / `1<сдвиг, длина>` | Флаг типа: литерал или ссылка |
| **LZ78** | `<позиция, символ>` | Динамический словарь, индекс + символ |
| **LZW** | `код (число)` | Только коды, предзаполненный алфавит |

## 🚀 Технологии

- **Vue 3** + Composition API + TypeScript
- **Pinia** для управления состоянием
- **Vue Flow** для визуализации деревьев
- **Vite** для сборки
- **GitHub Pages** для деплоя
- Адаптивный дизайн + тёмная/светлая тема

## Запуск локально

```bash
# Клонировать репозиторий
git clone https://github.com/koupcls/Calculator.git
cd Calculator/app

# Установить зависимости
npm install

# Запустить дев-сервер
npm run dev

# Сборка для продакшена
npm run build
```
